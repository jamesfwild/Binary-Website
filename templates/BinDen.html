Converting Denary to Binary

Here you will learn how to convert denary to binary through a step by step guide using an example.

Question: Convert 143 into Binary.

Step 1 - Write out the first 8 binary place values

1286432168421

Step 2 - Starting from the largest place value and see whether it can be subtracted from your value.

So can 128 be taken away from 143? 143-128 = 15. Yes, therefore we shall put a 1 in the 128 column.

1286432168421
1

Step 3 - Repeat step 2 for 64.

So can 64 be taken from 15? 15-64 won't work. So now we put a 0 in the 64 column.

1286432168421
10

Step 4 - Repeat step 2 again for 32.

So can 32 be taken from 15? 15-32 won't work. Therefore, put a 0 in the 32 column.

1286432168421
100

Step 5 - Then again for 16.

So can 16 be taken from 15? 15-16 again won't work. Therefore, put a 0 in the 16 place value column.

1286432168421
1000

Step 6 - Try again for 8.

So can 8 be taken from 15? 15-8=7. Yes! Therefore, put a 1 in the 8 column.

1286432168421
10001

Step 7 - Repeat again for 4.

So can 4 be taken from 7? 7-4=3. So therefore, we put a 1 in the 4 column.

1286432168421
100011

Step 8 - Repeat for place holder 2.

Can 2 be taken from 3? 3-2=1. Yes, so we put a 1 in the 2 column.

1286432168421
1000111

Step 9 - Finally, repeat for 1.

So can 1 be taken from 1? 1-1=0. Therefore, add a 1 in the 1 place value column.

1286432168421
10001111

Answer: 143 = 10001111

Converting Binary to Denary

Question: Convert 11010010 into Binary.

Step 1 - Write out the first 8 binary place values along with the corresponding binary value.

1286432168421
11010010

Step 2 - Starting from the largest place value look to see whether it has a value of 1 or 0.

As our first value is 1 add that to the total.

1286432168421
11010010
128+

Step 3 - Repeat step 2 for 64.

As this value is 1 add 64 to the total.

1286432168421
11010010
128+64+

Step 4 - Repeat this again for 32.

This value is 0 so do not add it to the total.

1286432168421
11010010
128+64+0+

Step 5 - Then again for 16.

This has a value of 1 so add this to the total.

1286432168421
11010010
128+64+0+16+

Step 6 - Try again for 8.

As this has a value of 0 do not add it.

1286432168421
11010010
128+64+0+16+0+

Step 7 - Repeat again for 4.

This has a value if 0 so do not add it.

1286432168421
11010010
128+64+0+16+0+0+

Step 8 - Repeat for place holder 2.

This has a value of 1 so add it to the total.

1286432168421
11010010
128+64+0+16+0+0+2+

Step 9 - Finally, repeat for 1.

As this is 0 do not add it.

1286432168421
11010010
128+64+0+16+0+0+2+0

Answer: 11010010 = 210
