Binary Shifts

Left Shifts

Left shifts occur when you multiply a binary number.

Note that when multiplying a binary number, if it is x2 shift it left by 1, if it is x4 shift it by 2, if it is x8 shift it by 3 and so on.

Question: 1101 x 2

1286432168421
1101

Now shift this left by 1.

1286432168421
11010

Now we should convert both of these values to see if we got the expected result.

1101 = 13

11010 = 26

This proves that our shift was correct.

Answer: 1101 x 2 = 11010

Right Shifts

Right shifts occur when you are dividing a number.

Remember to note that when dividing a binary number, if it is /2 shift it right by 1, if it is /4 shift it by 2, if it is /8 shift it by 3 and so on.

Question: 101100 / 4

1286432168421
101100

Now shift this right 2 times.

1286432168421
1011

Now double check this by converting these values to denary.

101100 = 44

1011 = 11

Answer: 101100/4 = 1011
