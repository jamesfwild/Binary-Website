Converting Binary to Hexadecimal

Before using this page ensure that you are comfortable with converting binary values to denary.

Here you will learn how to convert binary to hexadecimal through a step by step guide using an example.

Question: Convert 11010110 into Hexadecimal.

Step 1 - Split the binary number into groups of 4 and write first 4 place values above them

8421
1101

8421
0110

Step 2 - Convert these binary values into denary.

8421
1101
8+4+0+1

So this section adds up to 13.

8421
0110
0+4+2+0

So this section adds up to 6

Step 3 - Now we have to convert this to hexadecimal.

So we have to check whether any of our values are 10 or above.

If so these will have to be converted using the table below.

101112131415
ABCDEF

This means we will have to convert the 13 to D.

Now to get our answer we have to combine these values, so we get D6.

Answer: D6

If this was not clear please revisit how to convert binary to denary.

Converting Hexadecimal to Binary

Before learning about this ensure you are comfortable with converting denary to binary.

Question: Convert 7F into binary.

Step 1 - Convert any letters into numbers.

To do this we can look at the table below.

ABCDEF
101112131415

So this means that our F will be converted to 15. So we now have 7 15.

Step 2 - Split the hex into 2 and write out the first 4 place values for each.

Step 3 - Convert each of these values individually into binary.

Converting 7 into binary

8421
0111

Converting 15 into binary

8421
1111

Step 4 - Combine these values to get an 8 bit binary string

Answer: 01111111
