Introduction

Denary

This is the number system we use in everyday life. This is also known as base 10, as there are 10 digits from 0 to 9.

Binary

Binary is also known as in base 2, as it has 2 digits 0 or 1. This is how your place values are created, the first 8 shown below. We will be mostly working in 8 bit binary as this will be most commonly used within your exams.

1286432168421
2726252423222120

As we are working in 8-bit binary, the maximum denary number we can reach is 255. Otherwise, an error will occur.

Note that when working in binary rather than writing 00011001 you can instead write 11001. This is more beneficial to save time but do not get confused with their place values!

Most significant bit = the bit in a binary number of the highest value.

Least significant bit = the bit in a binary number of the lowest value.

Tip:

When working in binary if you notice that the least significant bit is equal to 1 then the denary number must be odd!

If it is equal to 0 then it must be even.


Character Sets

When you press a key on a keyboard, a number is generated that represents the symbol for that key. This is called a character code. A complete collection of characters is a character set.

ASCII and Unicode are 2 of the main examples that you will need to be aware of.

ASCII

ASCII is made up of 7-bit binary codes, meaning it can hold 128 characters

Also, make sure you are aware of Extended ASCII, which works with 8-bit binary, meaning it can hold 256 characters. This is useful when dealing with other European languages, which may make use of accents and use other symbols.

Unicode

Unicode uses between 8 to 32 bits per character, this means it can represent symbols from languages all across the world. This is often used on the internet, so that users can communicate in many different languages.

Hexadecimal

Hexadecimal is also known as base 16. The main purpose of hex is to simplify binary codes to make it more understandable for people.

Note that when working with hex the values can range from 0 to 15. However, when the value is 10 or over it is converted to a letter. The table below represents this.

ABCDEF
101112131415

This means that the smallest value you can represent is 00 and the largest is FF.

It is often used to represent colours in the form of #RRGGBB, where RR is reds, GG is greens and BB is blues. The # just represent that it has been written in the hex format.
