Binary Addition

Before starting there are 4 main rules you have to remember:

  1. 0 + 0 = 0
  2. 0 + 1 = 1
  3. 1 + 1 = 10 (binary for 2) meaning it equals 0 carry 1
  4. 1 + 1 + 1 = 11 (binary for 3) meaning it equals 1 carry 1

Question 1: What is 01100101 + 10010110

Step 1 - Set the question up in the form of long addition

01100101
+10010110

Step 2 - Starting from the right hand side add the 2 corresponding numbers.

As it is 1 + 0, it will equal 1.

01100101
+10010110
=1

Step 3 - Repeat step 2 but move left once.

As it is 0 + 1, it will equal 1 again.

01100101
+10010110
=11

Step 4 - Repeat this again and move left once more.

As it is 1 + 1, it will equal 0 and carry the 1.

01100101
+10010110
=011
1

Step 5 - Now do the same to the next column.

As it is 0 + 0 + 1, it will equal 1.

01100101
+10010110
=1011
1

Step 6 - Same again for the next column.

As it is 0 + 1, it will equal 1.

01100101
+10010110
=11011
1

Step 7 - Repeat this again for the all of the columns remaining.

01100101
+10010110
=11111011
1

Answer: 01100101 + 10010110 = 11111011


Question 2: What is 11010011 + 10011101

This question is much more challenging than the last so be prepared.

Step 1 - Starting from the far right add each column individually.

So 1 + 1 = 0 carry 1.

11010011
+10011101
=0
1

Step 2 - Move to the mext column and add up all values.

So 1 + 0 + 1 = 0 carry 1.

11010011
+10011101
=00
11

Step 3 - Repeat step 2.

So 0 + 1 + 1 = 0 carry 1.

11010011
+10011101
=000
111

Step 4 - Repeat that again.

So 0 + 1 + 1 = 0 carry 1.

11010011
+10011101
=0000
1111

Step 5 - Move left and add up the column.

So 1 + 1 + 1 = 1 carry 1.

11010011
+10011101
=10000
11111

Step 6 - Same again for the next column.

So 0 + 0 + 1 = 1 .

11010011
+10011101
=110000
11111

Step 7 - Repeat this again.

So 1 + 0 = 1 .

11010011
+10011101
=1110000
11111

Step 8 - Repeat this again.

So 1 + 0 = 1 .

11010011
+10011101
=101110000
111111

So here we now face an issue as our answer was 9 bits however both of our inputs were only 8 bits, this means that our value doesn't fit into the range of numbers that can be represented by 8 bits. This will cause an overflow error on our devices and it will not be able to store our answer.

Answer: Overflow error!
