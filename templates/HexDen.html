Converting Hexadecimal to Denary

Before using this page ensure that you are comfortable with converting hex to binary and binary to denary!

Here you will learn how to convert hexadecimal to denary through a step by step guide using an example.

Question: Convert D6 into Denary.

Step 1 - Convert any letters into numbers

To do this we can use the table below

ABCDEF
101112131415

So now we are left with 13 6.

Step 2 - Split the hex into 2 and write out the first 4 place values for each.

Step 3 - Convert each of these values into binary.

Converting 13 into binary

8421
1101

Converting 6 into binary

8421
0110

Step 4 - Combine these values to get an 8 bit binary string

Step 5 - Write out the first 8 place values

1286432168421
11010110

Step 6 - Convert this into denary

1286432168421
11010110
128+64+0+16+0+4+2+0

Answer: D6 = 214

If this was not clear please revisit converting hex to binary and binary to denary.

Converting Denary to Hexadecimal

Before using this page ensure that you are comfortable with converting denary to binary and binary to hex!

Question: Convert 124 into hexadecimal.

Step 1 - Write out the first 8 place values.

Step 2 - Convert this into binary

1286432168421
01111100

Step 3 - Split this in half and convert each section into hexadecimal

8421
0111
0+4+2+1

Adding these up you should get 7


8421
1100
8+4+0+0

From adding these up you should get 12 which according to the table below is equal to C.

101112131415
ABCDEF

Step 4 - Combine these values to get your answer.

Answer: 124 = 7C
